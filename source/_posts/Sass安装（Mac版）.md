> 1.确定是否安装了Ruby
```bash
$ruby -v
```
> 2.安装方法
- 安装方法一：
安装Sass
使用
```bash
sudo gem install sass
```
进行Sass安装
- 安装方法二：本地安装
1.在该网站https://rubygems.org/gems/sass
2.下载Sass版本
3.使用gem install 文件地址
```bash
$gem install /Users/***/Downloads/sass-3.4.23.gem 
```
3.使用sass -v 查看Sass版本号
```bash
sass -v
Sass 3.4.23 (Selective Steve)
```
